Creating a Workbook¶

In Metatron Discovery, a workbook functions as a standalone data analytics report. Once a workbook is created, you can store a number of dashboard slides in the workbook and present them in the proper order.

A workbook is created as follows:

  1. Click the + Workbook button at the bottom of the workspace to move to the workbook creation page.

  2. Enter a name (required) and description for the workbook to be created and click Done. If you select &ldquo;**Continue to create a dashboard of a new workbook**,&rdquo; you&rsquo;ll proceed directly to the &ldquo;**Create a Dashboard**&rdquo; page. This option is provided because a workbook cannot work without dashboards in it.

  3. After clicking the &ldquo;+ Add Data Source&rdquo; button in the middle of the screen, select a data source to create a dashboard. For more details on how to create a dashboard, refer to Creating a Dashboard.

  4. You can check the new workbook in the workspace home as shown below. Click the workbook to enter it.
