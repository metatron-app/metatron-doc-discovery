Introduction of Metatron Discovery¶

Metatron Discovery is a solution that analyzes data loaded into the Metatron server cluster in a simple, sophisticated manner, and visualizes the results in the user PC in the form of charts and reports. Comprised of web templates, the solution is highly accessible, and can be remotely accessed by registered users from any PC.

This section introduces the technical background and structure of Metatron Discovery, and the Druid engine powering Metatron.
