주요기능¶

Machine Learning¶

예측 모델에 기반한 알고리즘을 자동 추천하는 머신 러닝으로 이지 유즈

Alarm & Report¶

비정상적인 상황에 대한 알람을 즉각적으로 확인 하고 리포팅까지 이어진 서비스

Analyze¶
