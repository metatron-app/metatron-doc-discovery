데이터 스토리지 소개¶

metatron Discovery에서 사용할 데이터는 다음과 같은 두 가지 방식으로 연동시킬 수 있습니다.

  • SQL 데이터베이스와 직접 연동 (2.1절 참조)
  • 원천 데이터를 ‘Druid’라는 내부 엔진에 탑재하여 ‘데이터 소스’ 단위로 보관 (2.2절 참조)

데이터 스토리지는 이러한 데이터를 metatron에 연동하고 관리하는 기능을 제공합니다.
